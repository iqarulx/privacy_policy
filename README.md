# Privacy Policies Repository

This repository contains the hosted privacy policies for our distributed applications.

The privacy policies in this repository serve as the official documentation for how user data is collected, used and protected across our various applications.

## Purpose

- Central location for all app privacy policies
- Version control and change tracking
- Public accessibility for users
- Compliance with app store requirements

## Structure

Each application's privacy policy is stored in its own markdown file within this repository.

## Note

This repository is for privacy policy hosting only. For application code or other resources, please refer to their respective repositories.
